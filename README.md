# A reproduction of [Pomofocus](https://pomofocus.io/app) using Angular

### Demo

![](readme/demo.gif)

### Boilerplate

https://github.com/jm2097/angular-boilerplate

### Install dependencies

```sh
pnpm install # run `npm install -g pnpm` if you don't have pnpm installed
```

### Run project

```sh
pnpm dev
```
